  # Loan Prediction Using Machine Learning

## 📌 Project Overview
This project predicts whether a loan will be approved or not using Machine Learning classification algorithms.

## 📊 Dataset
- Loan Prediction Dataset
- Source: Kaggle

## 🧠 Algorithms Used
- Logistic Regression
- Decision Tree
- Random Forest
- K-Nearest Neighbors (KNN)

## 📈 Model Evaluation
- Accuracy score used for evaluation
- Best performing model identified based on accuracy

## 🛠 Tools & Technologies
- Python
- Jupyter Notebook
- Scikit-learn
- Pandas
- NumPy
